Mixeway Vault

About

Mixeway use HashiCorp Vault as secure storage. Passwords are hashed using BCrypt but there is need to use actual password or apiKey in it real form in order to log into Vulnerability Scanner. We have decided to not store it in plain text but use Vault instead.

Instalation and running

There is many tutorials how to get Vault up and running.

Use official HashiCorp Vault docummentationUse HashiCorp Vault Docker

Mixeway integration

In order to properly create Mixeway integrate with Vault to store and use passwords for Vulnerability scanner You have to edit env file and set properly:

VAULT_APP_NAME=<application name>
VAULT_SCHEME=<http | https>
VAULT_TOKEN=<vault token get during configuration>
VAULT_HOST=<IP or DNS for accessing Vault API>
VAULT_PORT=<Port on which Vault API is listening>
